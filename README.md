# 🌊 台灣避難所洪水風險評估系統 (ARIA)

## 📋 專案概述

ARIA (Automated Risk Assessment and Interactive Analysis) 是一個完整的台灣避難所洪水風險評估系統，整合了地理空間分析、風險分級、統計分析和多樣化視覺化功能。

### 🎯 主要功能

- **🧹 資料清理**: 自動移除無效座標的避難所資料
- **🗺️ 空間分析**: 使用 `gpd.sjoin` 進行精確的空間連接
- **⚠️ 風險分級**: 基於河川距離的四級風險評估
- **📊 統計分析**: 鄉鎮級別的風險統計和容量分析
- **🗺️ 視覺化**: 互動式地圖和靜態統計圖表
- **📋 審計輸出**: 完整的 JSON 風險審計檔案

### 📊 資料來源

1. **WRA 河川多邊形**: `data/RIVERPOLY/riverpoly.shp`
2. **消防署避難所**: `data/避難收容處所點位檔案v9.csv`
3. **TGOS 鄉鎮界線**: `data/鄉(鎮、市、區)界線1140318/TOWN_MOI_1140318.shp`

### 🎨 風險等級定義

| 風險等級 | 距離河川 | 顏色 | 說明 |
|---------|---------|------|------|
| 🔴 高風險 | 500m 以內 | 紅色 | 極高洪水威脅 |
| 🟠 中風險 | 500m-1km | 橙色 | 高洪水威脅 |
| 🟡 低風險 | 1km-2km | 黃色 | 中等洪水威脅 |
| 🟢 安全 | 2km 以上 | 綠色 | 低洪水威脅 |

## 🚀 快速開始


## 🔧 技術規格

### 座標系統
- **輸入**: WGS84 (EPSG:4326)
- **處理**: TWD97 (EPSG:3826) - 台灣適用投影座標系
- **輸出**: WGS84 (EPSG:4326) - 網頁地圖標準

### 主要套件
- **geopandas**: 地理空間資料處理
- **pandas**: 資料分析
- **folium**: 互動式地圖
- **matplotlib**: 靜態視覺化
- **shapely**: 幾何運算

### 空間分析方法
- **gpd.sjoin**: 空間連接用於鄉鎮分配
- **buffer**: 河川緩衝區生成
- **difference**: 互斥緩衝區建立
- **intersects**: 點與多邊形交集測試

#### 🚀 初始設定階段 (00:30-01:00)

- **結果**: 5,970 個有效避難所
- **座標欄位**: 確認經度(第5欄)、緯度(第6欄)

#### 🗺️ 空間分析優化 (01:15-01:30)
- **初始方法**: 直接幾何運算
- **優化**: 改用 `gpd.sjoin` 空間連接
- **鄉鎮分配**: 成功分配 5,907/5,970 個避難所
- **移除無效**: 63 個無法分配鄉鎮的避難所

#### 📊 風險分級改進 (01:30-01:45)
- **緩衝區重疊問題**: 發現互斥緩衝區需求
- **解決方案**: 建立獨佔緩衝區 (500m, 500-1000m, 1000-2000m)
- **安全分類**: 新增 "Safe" 類別給 2km+ 避難所
- **統計結果**: 高風險 2,568, 中風險 1,358, 低風險 1,163, 安全 818

#### 🎨 視覺化開發 (01:45-02:00)
- **互動地圖**: Folium 實現，包含河川、緩衝區、避難所點位

- **互動功能**: 點擊避難所顯示詳細資訊
- **圖例**: 清晰的風險等級說明

#### 📈 統計分析 (02:00-02:15)
- **鄉鎮分組**: 按鄉鎮統計避難所數量和容量
- **風險排名**: 基於安全容量比率排序
- **關鍵洞察**: 發現多個鄉鎮安全容量比率為 0%
- **前10名**: 平鎮區、山上區等最需要關注

#### 🔧 環境變數整合 (02:15-02:30)
- **設定檔案**: 建立 `.env` 環境變數配置
- **緩衝區參數**: BUFFER_HIGH=500, BUFFER_MED=1000, BUFFER_LOW=2000

#### 🐛 問題解決記錄

**問題1**: 緩衝區重疊導致重複計算
- **症狀**: 避難所被多次計入不同風險等級
- **原因**: 使用重疊緩衝區進行空間連接
- **解決**: 建立互斥緩衝區，確保每個避難所只屬於一個風險等級

**問題2**: 視覺化輸出檔案名稱衝突
- **症狀**: 互動式地圖被儲存為 PNG 檔案
- **原因**: `.env` 中 `OUTPUT_MAP=risk_map.png`
- **解決**: 分離 `OUTPUT_MAP` 和 `OUTPUT_CHART` 環境變數

**問題3**: 中文字體顯示問題
- **症狀**: matplotlib 圖表中文顯示為方塊
- **原因**: 缺少中文字體支援
- **解決**: 設定適當的字體和編碼

**問題4**: 河川圖層欄位名稱錯誤
- **症狀**: Folium 地圖無法載入河川工具提示
- **原因**: 使用錯誤的欄位名稱 'NAME'
- **解決**: 改用正確欄位 'RIVER_NAME'
